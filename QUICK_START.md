# Быстрый старт: Desktop версия с синхронизацией

## Вариант 1: Electron + Firebase (Рекомендуется)

### Преимущества:
- ✅ Работает как обычная программа на ПК
- ✅ Автоматическая синхронизация между 3 ПК в реальном времени
- ✅ Не требует собственного сервера
- ✅ Бесплатно для небольших проектов

### Шаги:

1. **Настройте Firebase** (5 минут):
   - Зайдите на https://console.firebase.google.com/
   - Создайте проект
   - Включите Realtime Database
   - Скопируйте конфигурацию

2. **Установите зависимости**:
   ```bash
   cd electron-setup
   npm install
   ```

3. **Настройте Firebase конфигурацию** в `firebase-config.js`

4. **Интегрируйте Firebase** в `script.js` (см. `INTEGRATION_GUIDE.md`)

5. **Соберите exe**:
   ```bash
   npm run build:win
   ```

6. **Установите на 3 ПК** - данные будут синхронизироваться автоматически!

---

## Вариант 2: Простое решение без Electron (только веб)

Если не хотите создавать exe, можно использовать веб-версию с Firebase:

1. Настройте Firebase (как в варианте 1)
2. Добавьте Firebase в `index.html` и `script.js`
3. Опубликуйте на GitHub Pages
4. Все трое работают через браузер, данные синхронизируются

---

## Вариант 3: Локальная версия + файловая синхронизация

Если не хотите использовать Firebase:

1. Используйте Electron (без Firebase)
2. Данные хранятся в файле на ПК
3. Используйте Dropbox/Google Drive/OneDrive для синхронизации файла
4. Каждый ПК читает/пишет в общую папку

⚠️ **Минусы**: Могут быть конфликты при одновременном редактировании

---

## Рекомендация

**Используйте Вариант 1 (Electron + Firebase)** - это самое надежное решение для вашей задачи.

### Почему:
- Работает как программа (не нужно открывать браузер)
- Автоматическая синхронизация без конфликтов
- Работает офлайн (с последующей синхронизацией)
- Бесплатно для вашего объема данных
