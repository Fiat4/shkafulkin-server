# Настройка доступа к редактированию

## Как изменить пароль

1. Откройте файл `script.js`
2. Найдите строку:
   ```javascript
   const AUTH_PASSWORD = 'admin123'; // ИЗМЕНИТЕ ЭТОТ ПАРОЛЬ!
   ```
3. Замените `'admin123'` на ваш пароль, например:
   ```javascript
   const AUTH_PASSWORD = 'ваш_надежный_пароль';
   ```
4. Сохраните файл и закоммитьте изменения

## Как работает аутентификация

- При первом открытии сайта пользователь увидит модальное окно с запросом пароля
- После ввода правильного пароля доступ к редактированию открывается на 24 часа
- Все функции редактирования защищены:
  - Добавление заказов
  - Редактирование заказов
  - Удаление заказов
  - Изменение статуса заказов
  - Импорт из Excel
  - Очистка всех заказов

## Важные замечания

⚠️ **ВНИМАНИЕ**: Это базовая защита на клиентской стороне. Пароль хранится в коде JavaScript, который виден всем, кто откроет исходный код страницы.

### Для более надежной защиты рассмотрите:

1. **Приватный репозиторий GitHub** (рекомендуется)
   - Сделайте репозиторий приватным
   - Дайте доступ только трем нужным людям
   - Это самый простой и надежный способ

2. **GitHub OAuth** (более сложно)
   - Интеграция с GitHub для аутентификации
   - Требует настройки OAuth приложения

3. **Внешний сервис аутентификации**
   - Firebase Auth
   - Auth0
   - Другие сервисы

## Распространение пароля

Сообщите пароль только трем авторизованным пользователям через безопасный канал (личное сообщение, а не публично).
